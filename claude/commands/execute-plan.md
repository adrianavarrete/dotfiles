# How to execute a plan

1. Ask the user for the plan to execute if not already specified.
2. Execute the actions of the current phase.
3. Check the checkboxes of the current phase to-do list that have been completed.
4. It is a MUST to ensure that the changes made are working before marking the actions as completed.
5. Update the plan next step section with the next phase to be completed.
6. Your final terminal output must suggest a commit message for all changes made in the current branch.
